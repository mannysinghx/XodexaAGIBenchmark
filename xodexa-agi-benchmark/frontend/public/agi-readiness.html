AGI Readiness Index
Live results — scored from real models. Looking for the illustrative sample? See the DEMO menu.
Important. Xodexa never declares a model "AGI." The highest attainable status on this ladder is "AGI-Level Candidate" (Level 5) — a model whose measured capability profile is consistent with general intelligence on the rotated private set, pending independent replication. "AGI" is not an output of this benchmark.
The AGI Readiness Index is a weighted blend of ten capability sub-scores. Each model is placed on a seven-step ladder from Level 0 (tool-like narrow model) to Level 6 (superhuman generalist candidate). The radar (or bars) shows where capability is broad versus where it gates the next level.
