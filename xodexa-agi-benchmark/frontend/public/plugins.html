Xodexa AGI Benchmark — Plugin Registry

Signed, sandboxed, default-deny extensions · benchmark packs, dataset adapters, tool simulators, safety packs
registry · Ed25519 required
Plugins extend Xodexa with new benchmark packs, dataset adapters, tool simulators and safety probes. Every plugin is signed (Ed25519), ships an SBOM + checksum, and runs under a default-deny policy: no network, sandbox-only filesystem, no secrets, no shell — unless explicitly granted and approved.
